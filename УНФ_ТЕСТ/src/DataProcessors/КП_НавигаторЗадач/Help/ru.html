Обработка "Навигатор задач"

Обработка "Навигатор задач" предназначена для вывода списков задач корпоративных бизнес-процессов. Форма обработки содержит два списка:

  • Список полученных задач. Данный список может быть сгруппирован по различным критериям. 
  • Список выданных текущим пользователем задач. Данный список может быть сгруппирован по исполнителю и по текущему состоянию задания.

В форму списка выводятся только те задания, к которым текущий пользователь имеет отношение, т.е. он является исполнителем, автором или контролером задания.

Обработка "Навигатор задач" включена в состав рабчего стола системы.
