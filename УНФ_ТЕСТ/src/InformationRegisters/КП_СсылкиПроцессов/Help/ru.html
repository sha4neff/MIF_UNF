Регистр сведений "Список ссылок бизнес-процессов"

Регист сведений "Список ссылок бизнес-процессов" предназначен для учета ссылок на объекты, прикрепленных к корпоративному бизнес-процессу. В качестве объектов (реквизит "Объект") могут выступать:

  • Корпоративные документы
  • Ссылки на внешние объекты
  • Файлы

В реквизите "Примечание" может быть сохранено примечание к указанному объекту. В реквизите "Бизнес-процесс" ссылка на бизнес-процесс, для которого приведен данный список ссылок.

Данные регистра сведений редактруются из формы корпоративного бизнес-процесса.
