Документ "План продаж"

Документ предназначен для хранения информации по предполагаемому объему продаж.

 

Планирование происходит в зависимости от выбранного сценария планирования.

 

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, для которой производится планирование продаж. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Подразделение. В данном реквизите указывается собственное подразделение, для которого производится планирование продаж. Выбирается из справочника "Структурные единицы" с типом "Административно-управленческое подразделение". Реквизит необязателен для заполнения.

  • Сценарий планирования. В данном реквизите указывается сценарий планирования, который определяет периодичность и основное измерение планирования. Выбирается из справочника "Сценарии планирования".

 

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования видов цен и валюты данного документа.

  • Запасы. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Объект планирования - конкретный тип зависит от указанного в сценарии измерения.

    • Характеристика - при необходимости в данном поле указывается характеристика планируемой для продажи номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Количество - количество планируемой номенклатурной позиции.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Цена - цена планируемой номенклатурной позиции.

    • Сумма - стоимость планируемой для продажи номенклатуры по данной строке.

    • Всего - поле, автоматически заполняемое суммарной стоимостью запасов согласно настройкам цен и валют. Настройки устанавливаются в диалоге, вызываемом по гиперссылке "Редактировать цены и валюту".

    • Заказ покупателя - источник потребности. Выбирается документ "Заказ покупателя". Реквизит необязателен для заполнения.

  • Дополнительная информация. Группа реквизитов:

    • Ответственный - информация о сотруднике, ответственном за ввод текущего документа. Выбирается из справочника "Сотрудники". Реквизит необязателен для заполнения.

    • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.
