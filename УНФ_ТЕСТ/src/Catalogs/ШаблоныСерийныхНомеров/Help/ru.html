Справочник "Шаблоны серийных номеров"

Справочник "Шаблоны серийных номеров" предназначен для хранения списка всех тех шаблонов, которые применяются для генерации серийных номеров в программе. Шаблоны этикеток и ценников могут быть добавлены непосредственно в форме подбора серийных номеров.

Пример:

Наберите в поле шаблона символы в том порядке, в каком они будут вводиться для серийного номера
•  Укажите # вместо цифр серийного номера, которые будут изменяться
•  Укажите префикс, суффикс или другие символы серийного номера в порядке их следования в серийном номере
Например: Для номера АА-000001 шаблон будет выглядеть так:АА-######
