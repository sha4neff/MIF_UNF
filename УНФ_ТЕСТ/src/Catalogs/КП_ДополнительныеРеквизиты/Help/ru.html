Справочник "Дополнительные реквизиты"

Справочник "Дополнительные реквизизиты" предназначен для учета реквизитов корпоративных бизнес-процессов и дополнительных реквизитов корпративных документов. Дополнительные реквизиты представляют собой удобный механизм для учета произвольных данных (переменных) и передаче их между исполнителями задач процесса, а так же для использования в корпоративных документах для хранения дополнительной информации.

Элементы справочника подчинены элементам справочника "Виды корпоративных процессов" и "Виды документов".
